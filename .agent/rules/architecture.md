ARCHITECTURE

General rules:
- Frontend-only application
- No backend
- No server
- No external APIs
- No user data persistence outside the browser

Technical stack:
- React
- TypeScript
- Vite
- Tailwind CSS

Architecture principles:
- Component-based structure
- Clear separation between:
  - UI components
  - application state
  - pure logic
- Prefer simple solutions over complex abstractions
- Avoid unnecessary dependencies
