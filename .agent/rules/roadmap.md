DEVELOPMENT ROADMAP

The project must be developed in phases.

Phase 1:
- Project scaffolding
- Tooling setup
- Basic layout

Phase 2:
- JSON input editor
- Real-time JSON validation

Phase 3:
- JSON tree viewer
- Expand / collapse behavior

Phase 4:
- Toolbar actions:
  - prettify
  - minify
  - copy
  - download

Phase 5:
- UI polish
- UX refinements

Phase 6:
- GitHub Pages deployment
- README documentation

Do not skip phases.
